# AI Course Assistant

## Problem
Students struggle to find answers from large course materials.

## Solution
An AI-powered assistant using RAG (Retrieval Augmented Generation)
that answers questions from course documents.

## Tech Stack
Python
Streamlit
Ollama
Sentence Transformers

## Features
- Ask questions from course PDFs
- Fast semantic search
- AI-generated answers

## How to Run

pip install -r requirements.txt
streamlit run app.py
